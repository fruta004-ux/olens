2. 진행상황 항목 삭제 + 프로젝트명 영역 확장

사용 빈도가 낮던 진행상황 컬럼(및 필터)을 제거하고, 확보한 공간만큼 프로젝트명 열 너비를 넓혔습니다.

3. 신규 명세서 알림 팝업 (재무 담당자 전용)

재무 담당자(role=finance) 계정으로 로그인하면, 새 명세서가 생길 때 화면 중앙 팝업으로 알립니다. 각 항목을 확인 버튼으로 처리해야 사라지며, 그냥 닫으면 새로고침 시 다시 떠서 누락을 방지합니다.
